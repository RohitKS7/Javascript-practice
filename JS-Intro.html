Modern JavaScript Intro

ECMAScript 6 or ES6 was introduced in 2015 that brought many major features in JavaScript. This features are the most widely used features of all time.

The following features were introduced in ES6 -

  • 1. Const and Let.
  • 2. Arrow Functions.
    const arrFun = () => {};
  • 3.Default Parameters. [John Doe is default parameter]
    const defPar = (name = 'John Doe') => {};
  • Template Strings =>
    const name = 'John Doe'
    console.log(`my name is ${name}`)
    result will be // my name is John Doe
